## ➡️ Installation

### 📥 Clone Project

```bash
git clone https://github.com/koyorin/docs.git koyodocs
cd koyodocs
```

### 📦 Install all required dependencies

```bash
yarn
```

### 🏃 Start development server

```bash
yarn start
```

### 🎉 Open http://localhost:3000 to see the result.
