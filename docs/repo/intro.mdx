# Introducción

Somos una organización independiente y estamos en GitHub. Actualmente, nuestra organización cuenta con 2 miembros, cada uno con su propio rol.
Inicialmente, el código estaba alojado en la cuenta de [@itzfrancojs](https://github.com/ItzFrancoJS), pero después de pensarlo mucho,
"para hacerlo más ordenado y facilitar la gestión, ¿por qué no transferir este repositorio a una organización?"
Así es como la organización [@neonixhub](https://github.com/neonixbot) se formó y se convirtió en una organización independiente.

## Roles

<!-- prettier-ignore-start -->

* [@itzfrancojs (Franco Cruz)](https://github.com/ItzFrancoJS) es el fundador y desarrollador principal de este proyecto.
* [@itzsilent (Yanixan Ayuzo)](https://github.com/itzsilent) trabaja en documentación y revisión de código del proyecto.

## Apóyanos

Actualmente usamos recursos y servicios gratuitos para mantener este bot en línea 24/7 en la medida de lo posible. Nos esforzamos para alcanzar un 99.9% de tiempo de actividad.
Sin embargo, no podemos garantizar que esté en línea para siempre con ese nivel de estabilidad, ya que seguimos usando un plan gratuito y existen muchas restricciones y limitaciones.
Si quieres apoyarnos, puedes hacerlo a través de los siguientes enlaces.

<!-- prettier-ignore-start -->

* Si tu moneda no es IDR (puede ser USD, JPY, etc.), puedes optar por donar con [Patreon](https://patreon.com/neonixhub).

<!-- prettier-ignore-end -->

¡Gracias por tu apoyo! El 97% de tus donaciones se utilizarán para las necesidades del proyecto (servidores de alojamiento). El 3% se destinará a necesidades relacionadas con la organización (dominios, servidores web, etc.).
