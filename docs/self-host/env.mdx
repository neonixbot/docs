# Variables

Las variables de entorno son un archivo donde almacenas la configuración de tu bot, como el Token, Prefijo y URL de la base de datos. No debes subir este archivo a tu repositorio y debes mantenerlo privado.

### Crear un archivo de variables de entorno

Crea un archivo `.env` a partir del archivo de ejemplo:

<Tabs>
<TabItem value="Linux / macOS">

```bash
cp .env.example .env
```

</TabItem>
<TabItem value="Windows">

```batch
copy .env.example .env
```

</TabItem>
</Tabs>

### Editar la configuración

El archivo de configuración se ha creado en `.env`. Edítalo con tu editor de código favorito y llena todos los campos requeridos.

```py title=".env"
# Variables principales
TOKEN="TU TOKEN DEL BOT"
PREFIX="TU PREFIJO"
MONGOURI="TU CADENA DE CONEXIÓN DE MONGO"

# Variables de clave API
HOLODEX_APIKEY="TU CLAVE API DE HOLODEX.NET"

# Variables de utilidad
TESTSERVER="ID DE TU SERVIDOR DE PRUEBAS"
OWNERID="ID DE TU CUENTA DE DISCORD"
ORIGIN_CHANNEL="ID DE TU CANAL DE ORIGEN"
UPLOAD_CHANNEL="ID DE TU CANAL DE SUBIDAS"
```

### Explicación

¿No estás seguro de cómo llenar estos valores? Aquí tienes una breve explicación sobre las variables de entorno.

#### `TOKEN`

El Token de tu bot en Discord. Puedes generar uno nuevo en [Discord Developer Portal](https://discord.com/developers/applications).

#### `PREFIX`

El prefijo de tu bot en Discord. Normalmente es de 1 o 2 caracteres especiales. Si usas `$` como prefijo, podrás ejecutar los comandos con `$` antes del nombre del comando.

> Ejemplo:
>
> `$help`

#### `MONGOURI`

La cadena de conexión URI de tu base de datos MongoDB. Ejemplo:

```
mongodb+srv://usuario:<contraseña>@clusterx.xxxx.mongodb.net/db-path
```

#### `TESTSERVER`

El ID de tu servidor de pruebas en Discord.

#### `OWNERID`

El ID de tu usuario en Discord.

#### `ORIGIN_CHANNEL`

El canal donde se subirán los enlaces. Por ejemplo, si usas `# 😳︱nsfw-copylink`, copia el ID del canal y reemplázalo en esta variable.

#### `UPLOAD_CHANNEL`

El canal donde se subirán los archivos adjuntos. Por ejemplo, si usas `# 😳︱nsfw-upload`, copia el ID del canal y reemplázalo en esta variable.

#### `HOLODEX_APIKEY`

Tu clave API de [Holodex](https://holodex.net). Se usa para obtener información sobre VTubers.
