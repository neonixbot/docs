# Installation

Bring the source code into your computer and install some required things to make this bot runs in your computer.

## Requirements

- [Node.js](https://nodejs.org) version >= 16 or above Installed
- [Yarn](https://yarnpkg.com) package manager Installed
- [Git](https://git-scm.com) or [GitHub CLI](https://cli.github.com/) Installed
- Your favorite code editor (such as [VS Code](https://code.visualstudio.com), [Atom](https://atom.io), etc)

## Clone the Repository

Clone the repository from GitHub and place it to your local computer

<Tabs>
<TabItem value="Git">

```bash
git clone https://github.com/gifaldyazkaa/koyorin.git
cd koyorin
```

</TabItem>
<TabItem value="GitHub CLI">

```bash
gh repo clone gifaldyazkaa/koyorin
cd koyorin
```

</TabItem>
</Tabs>

## Install all required dependencies

Install all required dependencies in [package.json](https://github.com/gifaldyazkaa/koyorin/blob/master/package.json) into your computer. You can use `yarn`, `pnpm`, or `npm`.

<Tabs>
<TabItem value="yarn">

```bash
yarn install
```

</TabItem>
<TabItem value="pnpm">

```bash
pnpm install
```

</TabItem>
<TabItem value="npm">

```bash
npm install
```

</TabItem>
</Tabs>
