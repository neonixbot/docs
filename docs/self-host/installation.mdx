# Instalación

Trae el código fuente a tu computadora e instala los requisitos necesarios para que este bot funcione en tu sistema.

## Requisitos

Se recomienda configurar [Node.js](https://nodejs.org) >= 16.12.0 con [yarn](https://yarnpkg.io).

## Clonar el repositorio

Clona el repositorio desde GitHub y guárdalo en tu computadora local.

<Tabs>
<TabItem value="Git">

```bash
git clone https://github.com/neonixbot/neonixbot.git
cd neonixbot
```

</TabItem>
<TabItem value="GitHub CLI">

```bash
gh repo clone neonixbot/neonixbot
cd neonixbot
```

</TabItem>
</Tabs>

## Instalar todas las dependencias requeridas

Instala todas las dependencias necesarias que se encuentran en [package.json](https://github.com/neonixbot/neonixbot/blob/master/package.json) en tu computadora. Puedes usar `npm`, `pnpm` o `yarn`.

<Tabs>
<TabItem value="npm">

```bash
npm install
```

</TabItem>
<TabItem value="pnpm">

```bash
pnpm install
```

</TabItem>
<TabItem value="yarn">

```bash
yarn install
```

</TabItem>
</Tabs>
