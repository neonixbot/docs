# Installation

Bring the source code into your computer and install some required things to make this bot runs in your computer.

## Requirements

A [node.js](https://nodejs.org) >= 16.12.0 setup with [yarn](https://yarnpkg.io) is highly recommended.

## Clone the Repository

Clone the repository from GitHub and place it to your local computer

<Tabs>
<TabItem value="Git">

```bash
git clone https://github.com/vzetaa/zeta.git
cd zeta
```

</TabItem>
<TabItem value="GitHub CLI">

```bash
gh repo clone vzetaa/zeta
cd zeta
```

</TabItem>
</Tabs>

## Install all required dependencies

Install all required dependencies in [package.json](https://github.com/vzetaa/zeta/blob/master/package.json) into your computer. You can use `yarn`, `pnpm`, or `npm`.

<Tabs>
<TabItem value="yarn">

```bash
yarn install
```

</TabItem>
<TabItem value="pnpm">

```bash
pnpm install
```

</TabItem>
<TabItem value="npm">

```bash
npm install
```

</TabItem>
</Tabs>
