# Iniciar el bot

## Servidor de producción

Inicia el bot en el servidor de producción.

<Tabs>
<TabItem value="yarn">

```bash
yarn start
```

</TabItem>
<TabItem value="dum">

```batch
dum start
```

</TabItem>
</Tabs>

## Servidor de desarrollo

Inicia el bot en el servidor de desarrollo. Este modo observa los archivos y reinicia el bot si se detectan cambios.

<Tabs>
<TabItem value="yarn">

```bash
yarn run dev
```

</TabItem>
<TabItem value="dum">

```batch
dum dev
```

</TabItem>
</Tabs>

:::info ¿Tienes un error?

Si has encontrado un error, consulta el [Repositorio en GitHub](https://github.com/neonixhub/neonixbot) y [crea un nuevo Issue](https://github.com/neonixhub/neonixbot/issues/new/choose), describiendo detalladamente el problema.

:::
