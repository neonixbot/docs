# Start the bot

## Production server

Start the bot in Production server

<Tabs>
<TabItem value="yarn">

```bash
yarn start
```

</TabItem>
<TabItem value="dum">

```batch
dum start
```

</TabItem>
</Tabs>

## Development server

Start the bot in Development server. Watch and restart if any changes was made.

<Tabs>
<TabItem value="yarn">

```bash
yarn run dev
```

</TabItem>
<TabItem value="dum">

```batch
dum dev
```

</TabItem>
</Tabs>
