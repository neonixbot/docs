# Estructura

```
/neonixhub <------- Carpeta Raíz
├── node_modules <- Dependencias del Proyecto
├── Commands <- Comandos del bot
   └── SlashCommands <-------- Comandos de SlashCommands (/)
      ├── Admin <- Comandos de Administración
      ├── Owner <- Comandos de Owner del bot
      ├── General <- Comandos generales
      ├── Music <- Comandos de Musíca
      └── Configs <- Comandos de configuración
   └── PCommands <---------- Comandos de Prefix (+)
      ├── Admin <- Comandos de Administración
      ├── Owner <- Comandos de Owner del bot
      ├── General <- Comandos generales
      ├── Music <- Comandos de Musíca
      └── Configs <- Comandos de configuración
├── Events <--------- Eventos del Bot
   ├── Ready <- Eventos de Inicio del bot
   └── Handlers <- Handlers de Eventos y Comandos del Bot
├── index.js <- Archivo Principal
├── config.js <- Configuración general del Bot
└── .env <- Archivo de configuración Importante
```
