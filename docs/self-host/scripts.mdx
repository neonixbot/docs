# Available Scripts

Available scripts that defined at [package.json](https://github.com/vzetaa/zeta/blob/master/package.json)

| Name        | Script   | Description                                |
| ----------- | -------- | ------------------------------------------ |
| Start       | `start`  | Start the bot                              |
| Development | `dev`    | Start the bot in Development server        |
| Build       | `build`  | Build the source code into JavaScript Code |
| Linters     | `lint`   | Lint the source code                       |
| Code Format | `format` | Format the source code                     |
