# User

User category. Dedicated to show information about user

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Avatar

Show user's avatar

```tsx
Command : !avatar
Aliases : !pfp
Usage   : !avatar [User?: User]
Example : !avatar @Falcxxdev#9273
```

### User information

Show user information

```tsx
Command : !userinfo
Aliases : !usr, !usrinfo
Usage   : !userinfo [User?: User]
Example : !userinfo @Falccxxdev#9273
```

## Slash Commands usage
