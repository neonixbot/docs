# VTubers

Get Channel Information, Upcoming stream, and Video metadata from VTuber's channel.

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Channel Info

Get VTuber's Channel Info.

```tsx
Command : !channelinfo
Aliases : !chinfo
Usage   : !chinfo <ChannelID: String>
Example : !chinfo UCTvHWSfBZgtxE4sILOaurIQ
```

### Upcoming Stream

Query and See Upcoming Stream from a VTuber's Channel.

```tsx
Command : !upcoming-stream
Aliases : !next-stream
Usage   : !next-stream <ChannelID: String>
Example : !next-stream UCTvHWSfBZgtxE4sILOaurIQ
```

### Video Info (Metadata)

Get Video Metadata by Video ID.

```tsx
Command : !videoinfo
Aliases : !vidinfo
Usage   : !vidinfo <VideoID: String>
Example : !vidinfo LS7ccVIIKdw
```
