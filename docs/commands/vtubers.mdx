# VTubers

Get Channel Information, Upcoming stream, and Video metadata from VTuber's channel.

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Channel information

Get VTuber's Channel Info.

```tsx
Command : !channelinfo
Aliases : !chinfo
Usage   : !chinfo <ChannelID: String>
Example : !chinfo UCTvHWSfBZgtxE4sILOaurIQ
```

### Upcoming stream

Query and See Upcoming Stream from a VTuber's Channel.

```tsx
Command : !upcoming-stream
Aliases : !next-stream
Usage   : !next-stream <ChannelID: String>
Example : !next-stream UCTvHWSfBZgtxE4sILOaurIQ
```

### Upload information

Get video or live uploads metadata by video id

```tsx
Command : !videoinfo
Aliases : !vidinfo
Usage   : !vidinfo <VideoID: String>
Example : !vidinfo LS7ccVIIKdw
```

## Slash Commands usage

### Channel information

Get VTubers channel info

```jsx
Command : /vtuber-channel <ChannelID: String>
```

### Upcoming stream

Get upcoming live stream of vtubers

```jsx
Command : /vtuber-next-stream <ChannelID: String>
```

### Upload information

Get video or live uploads metadata by video id

```jsx
Command : /vtuber-upload-info <VideoID: String>
```
