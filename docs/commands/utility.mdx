# Utility

List of Utility commands.

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Clear Message

Delete messages in specified amount

```tsx
Command : !clear
Aliases : !cls
Usage   : !clear <Amount: Number> // 5-100
Example : !clear 50
```

### GitHub User

Search for GitHub user by username

```tsx
Command : !github
Aliases : !gh
Usage   : !github <Username: String>
Example : !github gifaldyazkaa
```

### Say

Say a text

```
Command : !say
Aliases : !echo
Usage   : !say <Text: String>
Example : !say hi
```

### Urban Dictionary

Search slang word meanings from Urban Dictionary

```tsx
Command : !urban
Aliases : !urban-dict
Usage   : !urban <Word: String>
Example : !urban cunny
```
