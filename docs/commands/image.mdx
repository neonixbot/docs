## Normal usage

### Danbooru

Get an images from [Danbooru](https://danbooru.donmai.us).

```tsx
Command : !danbooru
Aliases : !booru
Usage   : !danbooru <Tags: String> // Max tags: 2
Example : !danbooru 1girl, ahoge
```

### Nekobot

Get an images from [Nekobot](https://nekobot.xyz)

```tsx
Command : !nekobot
Aliases : !nekob
Usage   : !nekobot <Type: String>
Example : !nekobot kemonomimi
```

### Nekos.life

Get an images from [Nekos.life](https://nekos.life)

```tsx
Command : !nekos
Aliases : !nekos-life, !nl
Usage   : !nekos <Type: String>
Example : !nekos neko
```

### Waifupics

Get an images from [Waifupics](https://waifu.pics)

```tsx
Command : !waifupics
Aliases : !waifupic
Usage   : !waifupics <Type: String>
Example : !waifupics waifu
```

### H-bomb

Continously made an request and send lewd images

```tsx
Command : !hentaibomb
Aliases : !setinterval
Usage   : !hentaibomb <Hostname: String> <Type: String> <Time: Number>
Example : !hentaibomb nekos.life lewd 1000
// This will sends lewd images from nekos.life every 1 second (1000ms).
// Infinity Loop, Until the Process is Destroyed.
```
