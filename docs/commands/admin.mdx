# Admin

Comandos de Administración del bot para mejorar tu servidor y su seguridad.

:::danger[NeonixHub Admin]
Algunos de estos comandos aún están en desarrollo.
:::

## Comandos

|  Nombre   |  Descripción          | Categoría  |
| --------- | --------------------- | -----------|
| Ban       | Banea un usuario      | Básico     |
| Unban     | Desbanea un usuario   | Básico     |
| Kick      | Expulsa un usuario    | Básico     |
| Mute      | Silenciar un usuario  | Básico     |
| Unmute    | Permitir hablar al usuario | Básico |
| softban   | Bloquea a un usuario de ver los canales por tiempo limitado | Básico |
