# Administrator

Bot Owner-only commands. Very powerful & Dangerous!

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Evaluate code

Evaluate given code.

```jsx
Command : !eval
Aliases : !e
Usage   : !eval <Code: CommonJS>
Example : !eval console.log(`Lolololol`);
```

### Terminal Emulator

Runs terminal command with ease.

```jsx
Command : !bash
Aliases : !sh, !zsh
Usage   : !bash <Command: UNIXCommand>
Example : !bash cat /etc/debian_version
```

### Kill Process

Kill current process & Destroy client session.

```jsx
Command : !kill
Aliases : !destroy
Usage   : !kill <Reasons: String>
Example : !kill Cuz im boring
```
