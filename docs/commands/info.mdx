# Information

Information category. Where all information served.

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Bot information

Show bot information

```
Command : !botinfo
Aliases : !info
Usage   : !botinfo
Example : !botinfo
```

### Discord.js documentation

Explore and search anything about discord.js in the docs!

```jsx
Command : !djs
Aliases : !discord.js
Usage   : !djs <Query: String>
Example : !djs client.user
```
