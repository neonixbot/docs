# Information

Information category. Where all information served.

## Notes

```yml
<> = Required value (Must be filled)
[] = Optional value (Can be filled or ignored)
```

## Normal usage

### Help Desk

Show help desk

```
Command : !help
Aliases : !h
Usage   : !help
Example : !help
```

### Ping

Show client ping latency

```
Command : !ping
Aliases : !p
Usage   : !ping
Example : !ping
```

### Up time

Show client uptime

```
Command : !uptime
Aliases : !up
Usage   : !uptime
Example : !uptime
```

### Bot Info

Show client information

```
Command : !botinfo

Usage   : !botinfo
Example : !botinfo
```
